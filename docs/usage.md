# Usage

Sardine offers a really simple CLI, with only two commands:

## Run a stack

```
sardine run <stack>
```

Arguments:

* `stack`: Name or alias of the stack you want to run.
* `-d or --detached`: (Optional) Runs the stack in detached mode.

## Stop a stack

```
sardine stop <stack>
```

Arguments:

* `stack`: Name or alias of the stack you want to run.
